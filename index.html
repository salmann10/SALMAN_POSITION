Position Absolute

Elemen dengan posisi absolute diposisikan relatif terhadap elemen terdekat yang memiliki posisi selain static (seperti relative, absolute, atau fixed). Jika tidak ada elemen seperti itu, maka elemen akan diposisikan relatif terhadap viewport. Untuk contoh penggunaan bisa melihat dalam box dibawah ini

Position Fixed

Elemen dengan posisi fixed diposisikan relatif terhadap viewport, sehingga tetap berada di tempat yang sama saat halaman digulir. Ini berguna untuk elemen seperti header atau footer yang ingin tetap terlihat. Untuk contoh penggunaan bisa melihat dalam box Header diatas.

Position Sticky

Elemen dengan posisi sticky berfungsi seperti relative pada posisi normal, tetapi akan beralih ke posisi fixed ketika mencapai batas tertentu saat menggulir. Ini sering digunakan untuk membuat elemen tetap terlihat saat pengguna menggulir halaman. Untuk contoh penggunaan bisa melihat dalam box dibawah ini
